# 90 · Evidence(证据,非规范)读法

> 存调研、原始讨论、外部研究。**解释"为什么设计成这样",但不能直接覆盖 canonical 正文。**

## 铁律

- **证据不等于决策。** 一份调研、一段对话,不代表它已压过之前的决策。
- **要"晋升"才生效**:证据若推翻某设计,先把结论正式写进 foundation / contracts / decision-log,再改正文和实现。晋升前的证据不能引用为"我们就这样定了"。
- 原始对话可保留完整,不要求像合同一样精简。

## 典型

`clarification-log.md`(循环追问留痕)、外部研究 dossier、原始讨论记录。
