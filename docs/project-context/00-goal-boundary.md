# 00 · 目标边界(Goal Boundary)

> 本文件是循环 Phase 0–1 的落点。下面每节先空着,讨论清楚一节就填一节,并更新 README 状态表。
> 未填的节保留标题 + `(待填)`,这是"还没讨论到",不是占位壳。

## 北极星(North Star)

(待填 — 一句话:为谁、解决什么核心问题。一句话能复述才算合格。)

## 设计箴言(Design Maxim)

(待填 — 一两句口诀式取舍,争论时的引力,能在评审里否决一个 PR。)

## Done(外部可观察)

(待填 — 做完指"世界多了什么",不是"代码写完";每条能被没参与的人用动作验证。)

## Non-goals(非目标)

(待填 — 明确不做什么 + 为什么不做。对抗引擎的主战场。)

## Constraints(硬约束)

(待填 — 不可动的:技术栈 / 性能 / 安全 / 公私边界。)

## Stop / Escalate(停止条件)

(待填 — 什么情况必须停下来问人,而非继续。用户几乎从不主动说。)

## Autonomy(自主边界)

(待填 — 哪些实现者可自行拍板;哪些必须人确认。)
